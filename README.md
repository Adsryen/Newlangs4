# Newlangs4

## 新功能

- 全新界面：基于 VUE 框架，前后端分离、低耦合度
- 支持 API：NewLangs 有一套独立的 API 系统，您可以通过调用 API 使用或搭建分站
- 支持 TXT 解析：您可以通过 TXT 解析来验证您的域名所有权
- 支持手机验证码注册
- 支持kldns拥有的全部功能
